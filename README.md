# Lovely Bunnys

- This is a website where you can go and buy bunnys

## User Stories

As a user, I want to:
-	Log In
-	Sign Up
-	Logout
-	See available bunnys.
-	Create New Bunnys
- Edit bunnys info
- Delete bunnys
- click on bunny too see more info
- add/edit or delete snacks
- includes multiple informational pages (shipping, policy ext)
